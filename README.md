# Instagram Feed System using Linked List

## Overview
This project is a console-based Instagram Feed Management System developed using C++ and Linked List Data Structures.

The project demonstrates the implementation of:
- Singly Linked List
- Doubly Linked List

The system simulates how social media feeds manage posts using insertion, deletion, traversal, searching, and counting operations.

---

## Features

### Singly Linked List
- Insert Post at Beginning
- Insert Post at End
- Insert Post at Given Position
- Delete Post from Beginning
- Delete Post from End
- Delete Post from Given Position
- Search Post by Username
- Count Total Posts
- Display All Posts

### Doubly Linked List
- Forward Traversal
- Backward Traversal
- Insert Operations
- Delete Operations
- Search Operations
- Count Posts
